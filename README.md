# Sales-Analysis-Dashboard-Power_BI
**Tool:** Tableau  
**Description:**  
Sales Analysis Dashboard | Power BI Developed an interactive Power BI dashboard to analyze sales performance and key KPIs including total revenue, sales trends, product-wise and region-wise performance. Used Power Query for data cleaning and transformation, created DAX measures, and implemented slicers and filters.
